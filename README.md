# Internship_Code
Weather Application Dashboard Visualization System
This project develops a Weather Application Dashboard
Visualization System to help a retail client in the outdoor
sports industry analyze the impact of weather on product
sales. Using Python 3.8+, SQLite, and Power BI,
historical weather data from ten major U.S. cities and
corresponding retail sales records were collected,
cleaned, and merged into a unified dataset. The
combined data was stored in SQLite and linked to Power
BI for interactive visualizations. Five key analyses were
implemented, including identifying maximum and
minimum sales temperature ranges, best- and leastselling product categories, and location-wise sales trends,
enabling data-driven business decisions.
